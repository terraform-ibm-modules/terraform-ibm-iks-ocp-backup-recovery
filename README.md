# IBM Cloud Kubernetes/OpenShift Backup Recovery Module

This Terraform module enables backup and recovery for IBM Cloud Kubernetes Service (IKS) and Red Hat OpenShift on IBM Cloud clusters using IBM Backup Recovery Service.

## 📖 Protection Group Configuration Guide

For detailed guidance on configuring Kubernetes Protection Groups, including limitations, best practices, and working examples, see:

**[Protection Group Configuration Guide](./ProtectionGroup.md)**

This comprehensive guide covers:

- Key limitations and restrictions for Kubernetes protection groups
- Supported features and configuration options
- Complete configuration examples
- Troubleshooting common errors
- Best practices for production deployments
